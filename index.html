Custom Scratch Extensions

  • windtools.js

    Blocks that can change the X and Y position of the window, and change its height and width. Try it out here!

    There's a bug that slows down projects, to fix this, put a "wait (0) seconds" block after the windtools.js block.

    This only works with Electron.

  • urlopen.js

    Opens URLS on the default Browser. Try it out here!

    This only works with Electron.

  • alert.js

    Alert windows that popup. Try it out here!

    This only works with Electron.

  • getinfo.js

    It gets your browser, and OS information, into a reporter. Try it out here!

    This works everywhere.

  • embedpage.js

    A Scratch Extension that embeds a page. Try it out here!

    This only works with Electron.
